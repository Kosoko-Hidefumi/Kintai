# 研修医マスタ.csv からカード型研修医一覧を作成 — AI実装プロンプト

他のAIが研修医マスタ.csvからカード型の研修医一覧を作成する際に使用するプロンプトです。そのままコピーして利用できます。

---

## プロンプト（コピー用）

```
研修医マスタ.csv を読み込み、カード型の研修医一覧画面を作成してください。

### 入力データ（研修医マスタ.csv）

UTF-8またはShift-JISのCSVファイル。以下の列を含む（列名は日本語）：

| 列名 | 説明 | 必須 |
|------|------|------|
| 年度 | 年度（例: 2024） | ○ |
| 学年 | 学年（例: PGY1, PGY2） | ○ |
| 初・後 | 研修期（例: 47期初期, 50期後期） | ○ |
| PHS | 個人番号・ID | - |
| 名前 | 氏名（漢字） | ○ |
| ふりがな | 氏名の読み仮名 | ○ |
| 専門科 | 診療科（例: 内科, 外科） | ○ |
| 進路 | 進路状況 | - |
| 動向調査 | 動向 | - |
| 本籍 | 本籍地 | - |
| 出身大学 | 出身大学名 | - |
| 備考 | 備考 | - |

※ 列名が「初・後」でない場合、「初」「後」「期」を含む列名を探して使用すること。
※ 欠けている列は空文字として扱う。

### 出力要件：カード型一覧

1. **レイアウト**
   - カードをグリッド表示（1行あたり3〜4枚、レスポンシブ対応）
   - 各カードは白背景、角丸、影付き、ホバー時わずかに浮き上がる
   - カード上辺に専門科ごとの色帯（3px程度）を表示

2. **各カードの表示内容**
   - **上部**: 名前（太字・大きめ）、ふりがな（小さめ・グレー）
   - **右上**（あれば）: PHSを電話アイコン付きで表示
   - **タグ群**（flex-wrapで折り返し）:
     - 年度（あれば）: 「〇〇年」形式、グレー背景
     - 専門科: カラータグ（白文字）、専門科ごとに色を割り当て
     - 初・後: グレー背景で表示（例: 47期初期）
     - 出身大学（あれば）: グレー背景

3. **専門科の色分け**
   - 内科=青、外科=赤、産婦人科=ピンク、小児科=紫、救急=オレンジ、総合診療=緑 など
   - 未定義の専門科はグレー系で統一

4. **機能**
   - 学年（PGY1, PGY2 等）でタブまたはフィルタで絞り込み
   - カードクリックで詳細モーダル表示（全項目を表示）
   - ふりがな順でソート

5. **技術スタック（推奨）**
   - React + TypeScript、または Vue、または Vanilla JS
   - CSV読み込み: FileReader + 手動パース、または papaparse など
   - スタイル: Tailwind CSS または 同等のユーティリティCSS

### カードのUI例（テキスト）

    ┌─────────────────────────────────┐
    │ 相羽拓人              📞 2088   │
    │ あいば たくと                   │
    │                                 │
    │ [2024年] [内科] [47期初期]      │
    │ [琉球大学]                      │
    └─────────────────────────────────┘

### 注意事項
- CSVの1行目はヘッダーとして扱う
- 空行はスキップ
- 文字コードの自動検出（BOM対応）を行うこと
```

## 参考：既存実装

kintaiプロジェクト内の研修医一覧（resident_dashboard）は、同じ仕様で Excel（.xlsx/.xlsm）を読み込み、カード表示しています。

- **表示形式**: PersonCard コンポーネント（React）
- **データ構造**: 年度・学年・初・後・PHS・名前・ふりがな・専門科・進路・出身大学 等
- **専門科カラー**: 内科=#3B82F6, 外科=#EF4444, 産婦人科=#EC4899, 小児科=#8B5CF6, 救急=#F97316, 総合診療=#10B981 など

CSVの列構成を上記テーブルに合わせれば、同じUIで表示できます。
