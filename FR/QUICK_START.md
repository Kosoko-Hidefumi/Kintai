# クイックスタートガイド

## 5分で始める

### 1. 環境の準備（初回のみ）

```powershell
# venv環境を有効化
cd python/streamlit
.\venv\Scripts\Activate.ps1

# 必要なライブラリがインストールされているか確認
python -c "import streamlit; import pandas; import plotly; print('OK')"
```

### 2. アプリの起動

```powershell
# FRディレクトリに移動
cd ..\..\FR

# Streamlitアプリを起動
streamlit run app.py
```

ブラウザが自動的に開き、`http://localhost:8501`でダッシュボードが表示されます。

### 3. 基本的な使い方

1. **サイドバーでフィルターを設定**
   - 期間プリセットから選択（全期間、今月、先月、四半期、年間）
   - カテゴリやベンダーを選択
   - 支出金額の範囲を設定

2. **タブを切り替えて分析**
   - 📊 概要ダッシュボード: 全体の概要を把握
   - 📅 時系列分析: トレンドを確認
   - 📁 カテゴリ分析: カテゴリ別の詳細
   - 🏢 ベンダー分析: ベンダー別の詳細
   - 💰 支出分析: 支出パターンの分析
   - 📆 月別比較分析: 月次比較
   - 🔗 カテゴリ×ベンダー分析: クロス分析
   - 🔍 詳細データ探索: データの検索・エクスポート

3. **データのエクスポート**
   - 詳細データ探索ページからCSVまたはExcel形式でダウンロード

## トラブルシューティング

### ポート8501が使用中の場合

```powershell
streamlit run app.py --server.port=8502
```

### データが表示されない場合

- CSVファイル（`Logistic.csv`）がFRディレクトリに存在するか確認
- ブラウザのコンソール（F12）でエラーを確認

### グラフが表示されない場合

- フィルター条件を確認（データが存在する範囲か）
- ブラウザをリロード（F5）

## 主な機能

### フィルター機能
- ✅ 期間フィルター（日付範囲 + プリセット）
- ✅ カテゴリフィルター（複数選択）
- ✅ ベンダーフィルター（複数選択）
- ✅ 支出金額フィルター（範囲指定）

### 可視化機能
- ✅ 8種類の分析ページ
- ✅ インタラクティブなグラフ（Plotly）
- ✅ ヒートマップ、箱ひげ図、エリアチャートなど

### データエクスポート
- ✅ CSV形式
- ✅ Excel形式
