# Streamlitアプリの起動確認方法

## 現在の状況

ポート8502が既に使用中です。これは、アプリが既に起動している可能性があります。

## 確認方法

### 1. ブラウザでアクセス

以下のURLをブラウザで開いてください：

- **ポート8501**: http://localhost:8501
- **ポート8502**: http://localhost:8502
- **ポート8503**: http://localhost:8503

### 2. 使用中のポートを確認

**PowerShellで実行：**
```powershell
netstat -ano | findstr ":850"
```

### 3. Streamlitプロセスを確認

**PowerShellで実行：**
```powershell
Get-Process | Where-Object {$_.ProcessName -like "*streamlit*" -or $_.ProcessName -like "*python*"}
```

### 4. 既存のプロセスを終了して再起動

**既存のStreamlitプロセスを終了：**
```powershell
# プロセスIDを確認
netstat -ano | findstr ":8501"
netstat -ano | findstr ":8502"

# プロセスを終了（PIDは実際のプロセスIDに置き換える）
taskkill /PID <プロセスID> /F
```

**その後、再度起動：**
```powershell
.\venv\Scripts\Activate.ps1
streamlit run app.py
```

## 正常に起動している場合

ブラウザで以下のような画面が表示されます：

- サイドバーに「📅 勤怠管理システム」のタイトル
- ユーザー選択のセレクトボックス
- メニュー（カレンダー、休暇申請、イベント、掲示板）

## エラーが表示される場合

### エラー: "スプレッドシートIDが設定されていません"

1. サイドバーで「管理者」を選択
2. パスワードを入力して認証
3. 「設定（管理者専用）」セクションでスプレッドシートIDを確認

### エラー: "APIのレート制限に達しました"

1〜2分待ってから再度アクセスしてください。

### その他のエラー

エラーメッセージの全文をコピーして、確認してください。
