# Streamlitアプリへの正しいアクセス方法

## 問題

「153.246.172.126 からの応答時間が長すぎます」というエラーが出る場合、**Network URL**にアクセスしようとしている可能性があります。

## 解決方法

### 1. Local URLを使用する

Streamlitアプリは、ターミナルに以下のようなメッセージを表示します：

```
You can now view your Streamlit app in your browser.

Local URL: http://localhost:8501
Network URL: http://153.246.172.126:8501
```

**必ず「Local URL」を使用してください。**

### 2. 正しいURL

ブラウザのアドレスバーに以下を入力：

```
http://localhost:8501
```

または

```
http://127.0.0.1:8501
```

### 3. Network URLを使わない理由

- Network URLは他のデバイスからアクセスするためのものです
- 同じPCからアクセスする場合は、Local URLを使用します
- Network URLはファイアウォールやネットワーク設定によってアクセスできない場合があります

## 確認方法

### ターミナルに表示されているURLを確認

ターミナルに以下のようなメッセージが表示されているはずです：

```
You can now view your Streamlit app in your browser.

Local URL: http://localhost:8501
Network URL: http://153.246.172.126:8501
```

この場合、**http://localhost:8501** を使用してください。

### ブラウザでアクセス

1. ブラウザを開く（Chrome、Edge、Firefoxなど）
2. アドレスバーに `http://localhost:8501` と入力
3. Enterキーを押す

## トラブルシューティング

### エラー: "このサイトにアクセスできません"

**原因：** アプリが起動していない、またはポートが使用されていない

**解決方法：**
1. ターミナルでアプリが起動しているか確認
2. エラーメッセージがないか確認
3. 以下のコマンドでポートを確認：
   ```powershell
   netstat -ano | findstr ":8501"
   ```

### エラー: "接続がタイムアウトしました"

**原因：** Network URLにアクセスしようとしている

**解決方法：**
- `http://localhost:8501` を使用してください
- `http://153.246.172.126:8501` は使用しないでください

### アプリが起動しない場合

1. 仮想環境を有効化：
   ```powershell
   .\venv\Scripts\Activate.ps1
   ```

2. アプリを起動：
   ```powershell
   streamlit run app.py
   ```

3. ターミナルに表示された「Local URL」をコピー

4. ブラウザでそのURLを開く

## まとめ

- ✅ **使用するURL**: `http://localhost:8501`
- ❌ **使用しないURL**: `http://153.246.172.126:8501`（Network URL）

同じPCからアクセスする場合は、必ずLocal URLを使用してください。
