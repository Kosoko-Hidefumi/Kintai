# Cursor 指示テンプレ

## ルール
- 変更はPR単位。TASKS.mdの順に。
- 受け入れ条件を満たすことを最優先。
- 難しい最適化・過剰な抽象化はしない。
- 例外時はユーザーに分かるエラーメッセージを出す。

## PR3（スキャン）の指示例
目的:
- /scan でカメラを起動して EAN-13 を読み取り、products.jan に一致する商品へ遷移する

要件:
- zxing-js/browser を使う
- 連続読取を防ぐ（同じコードは1秒間無視など）
- 見つからない場合はトースト表示
- 見つかったら /products/[id] へ

受け入れ:
- 実機またはPCカメラでJANを読み取ったら商品詳細が開く
