# データ分析ダッシュボード

顧客購買データを可視化・分析するためのインタラクティブなダッシュボードウェブアプリケーションです。

## 技術スタック

- **React** 18.x
- **Material-UI (MUI)** v5
- **Recharts** - データ可視化
- **PapaParse** - CSVパーサー
- **Day.js** - 日付処理
- **Vite** - ビルドツール

## セットアップ

### 必要な環境

- Node.js 18.x以上
- npm、yarn、またはpnpm

### インストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:3000` を開きます。

### ビルド

```bash
npm run build
```

## 使用方法

1. アプリケーションを起動します
2. 「CSVファイルを選択」ボタンをクリックします
3. `sample-data.csv` ファイルを選択します
4. データが読み込まれたら、フィルタを使用してデータを絞り込みます
5. KPIカード、グラフ、テーブルでデータを分析します

## プロジェクト構造

```
src/
├── components/        # Reactコンポーネント
│   ├── Layout/       # レイアウトコンポーネント
│   ├── DataLoader/   # データ読み込みコンポーネント
│   ├── Filters/      # フィルタコンポーネント
│   ├── KPI/          # KPIカードコンポーネント
│   ├── Charts/       # グラフコンポーネント
│   └── Table/        # テーブルコンポーネント
├── contexts/         # React Context（状態管理）
├── utils/            # ユーティリティ関数
├── theme/            # MUIテーマ設定
└── App.jsx           # メインアプリケーションコンポーネント
```
