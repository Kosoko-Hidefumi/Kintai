# 研修医マスターデータ 期別リスト集計プロンプト

このプロンプトを使用することで、他のAIでも研修医マスターファイルから同様の期別集計を実行できます。

---

## タスク概要

「研修医マスタ.xlsm」ファイル（mainシート）から研修医データを読み込み、期ごと（47期〜59期）に分類・集計を行います。

---

## 入力データの形式

### ファイル形式
- ファイル名: `研修医マスタ.xlsm`
- シート名: `main`
- ヘッダー行: 「年度」と「名前」を含む行を自動検出

### 必須列
| 列名 | 説明 |
|------|------|
| 年度 | 記録の年度（数値） |
| 学年 | 研修医の学年 |
| 名前 | 研修医の名前 |
| 進路 | 現在の状態（研修中/転出/修了/中断/退職/進級など） |
| 初・後（または「期」を含む列） | 研修期（例: "47期初期"、"50期後期"） |
| 本籍 | 出身地/本籍地 |
| 動向調査 | 転出先・進路先の施設名 |

### オプション列
- ふりがな、性別、専門科、出身大学、備考、PHS

---

## 処理手順

### ステップ1: データの読み込みとクレンジング

1. mainシートからデータを読み込む
2. 「年度」と「名前」を含む行をヘッダー行として検出
3. 空白列（Unnamed列）と空白行を削除

### ステップ2: 名前の正規化（括弧名の統合）

同一人物が異なる名前で記録されている場合を統合します。

**ルール**: 名前に含まれる括弧（全角・半角）とその中身を削除

```
例:
- '筧咲陽子(渡部)' → '筧咲陽子'
- '山田花子（旧姓：鈴木）' → '山田花子'
```

### ステップ3: 期番号の抽出

「初・後」列または「期」を含む列から期番号を抽出します。

```
例:
- "47期初期" → 47
- "50期後期" → 50
- "53期" → 53
```

対象期: **47期〜59期**

### ステップ4: 各期ごとの最終記録の選択

同一人物（正規化後の名前）で複数の記録がある場合、以下のルールで最終記録を選択:

1. **終了進路（転出/修了/中断/退職）の記録がある場合**: その中で年度が最大のもの
2. **終了進路がない場合**: 年度が最大のもの

### ステップ5: 進路の分類

「進路」列の値を以下のように分類:

| 進路の値 | 分類結果 |
|----------|----------|
| 転出 | 転出 |
| 修了 | 修了 |
| 中断 | 中断 |
| 退職 | 退職 |
| 空白/進級/その他 | 研修中 |

### ステップ6: 出身地の判定（沖縄出身かどうか）

「本籍」列の値を使って判定:

**沖縄出身と判定するキーワード**:
- 沖縄
- おきなわ
- OKINAWA
- okinawa

これらのキーワードが本籍に含まれていれば「沖縄出身」、含まれていなければ「沖縄外出身」

### ステップ7: 転出先施設の判定（沖縄県内かどうか）

「動向調査」列の施設名を使って判定:

**沖縄県内施設と判定するキーワード**:
```
県立、宮古、北部、八重山、中部、南部、琉大、琉球大学、
伊平屋、伊是名、西表、小浜、座間味、阿嘉、大原、粟国、
渡名喜、波照間、北大東、南大東、精和、浦添、沖縄
```

これらのキーワードが施設名に含まれていれば「沖縄県内」、含まれていなければ「沖縄県外」

**沖縄県内の主要施設リスト（参考）**:
- 県立宮古病院、県立北部病院、県立八重山病院、県立中部病院
- 琉球大学医学部附属病院（琉大病院、琉大附属病院）
- 南部医療センター
- 浦添総合病院
- 各種離島診療所（伊平屋、伊是名、西表、小浜、座間味、阿嘉、大原、粟国、渡名喜、波照間、北大東、南大東など）

---

## 集計カテゴリ

各期の研修医を以下の7カテゴリに分類してカウント:

| カテゴリ | 条件 |
|----------|------|
| **研修中** | 進路が「研修中」（空白/進級/その他）の人 |
| **沖縄出身 → 沖縄内（転出・修了）** | 沖縄出身で、沖縄県内施設に転出・修了した人 |
| **沖縄出身 → 沖縄外（転出・修了）** | 沖縄出身で、沖縄県外施設に転出・修了した人 |
| **沖縄外出身 → 沖縄内（転出・修了）** | 沖縄県外出身で、沖縄県内施設に転出・修了した人 |
| **沖縄外出身 → 沖縄外（転出・修了）** | 沖縄県外出身で、沖縄県外施設に転出・修了した人 |
| **中断** | 進路が「中断」の人 |
| **退職** | 進路が「退職」の人 |

## 実行例プロンプト（他のAI向け）

以下のプロンプトをコピーして使用できます：

```
あなたは研修医データの集計を行うデータアナリストです。

以下の手順で「研修医マスタ.xlsm」ファイルのmainシートからデータを集計してください。

【データの読み込み】
1. mainシートを読み込む
2. 「年度」と「名前」を含む行をヘッダーとして検出
3. 空白列・空白行を削除

【名前の正規化】
名前に含まれる括弧（全角・半角）とその中身を削除して同一人物を統合
例: '山田花子(旧姓)' → '山田花子'

【期の抽出】
「初・後」または「期」を含む列から数字を抽出（47期〜59期が対象）

【最終記録の選択】
同一人物で複数記録がある場合:
- 終了進路（転出/修了/中断/退職）があればその中で年度が最大のもの
- なければ年度が最大のもの

【分類ルール】

■進路の分類:
- 転出/修了 → そのまま
- 中断 → 中断
- 退職 → 退職
- その他 → 研修中

■沖縄出身の判定（本籍列）:
「沖縄」「おきなわ」「OKINAWA」のいずれかを含む → 沖縄出身

■沖縄県内施設の判定（動向調査列）:
以下のキーワードを含む → 沖縄県内:
県立、宮古、北部、八重山、中部、南部、琉大、琉球大学、
伊平屋、伊是名、西表、小浜、座間味、阿嘉、大原、粟国、
渡名喜、波照間、北大東、南大東、精和、浦添、沖縄

【集計カテゴリ】
1. 研修中
2. 沖縄出身 → 沖縄内（転出・修了）
3. 沖縄出身 → 沖縄外（転出・修了）
4. 沖縄外出身 → 沖縄内（転出・修了）
5. 沖縄外出身 → 沖縄外（転出・修了）
6. 中断
7. 退職

各期（47期〜59期）ごとに上記7カテゴリの人数と該当者名を集計してください。
```

---

## 注意事項

1. **名前の括弧は旧姓などを示す**: 括弧内は削除して同一人物として統合
2. **転出と修了は同じカテゴリで集計**: 「転出・修了」としてまとめる
3. **動向調査が空の場合**: 転出先不明として「沖縄県外」扱い
4. **本籍が空の場合**: 「沖縄外出身」扱い
5. **期番号の抽出**: 「47期初期」「50期後期」などから数字部分のみ抽出
